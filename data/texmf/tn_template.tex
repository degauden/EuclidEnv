Note the use of gaia\_aa for the bib style. This allows the use of citell in
place of cite for livelink docs using this put the DOC code as the ref
rather than a number or author/year e.g. 

\begin{verbatim}
\citell{LL:WOM-011}.
\end{verbatim}

Other paper citations should just use cite and will come out AA style
e,g, 

\begin{verbatim}
\cite{omu01}.
\end{verbatim}
